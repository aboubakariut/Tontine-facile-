Tutoriel

Premiers pas dans Tontine Facile

Étape 1 : Créer votre compte

1. Cliquez sur "Inscription"
2. Entrez votre numéro WhatsApp et un mot de passe
3. Validez avec le code reçu par SMS (simulation pour l’instant)

Étape 2 : Créer votre première tontine

1. Depuis le dashboard → "+ Créer une tontine"
2. Choisissez nom, montant, fréquence, nombre de membres
3. Validez et invitez vos proches via WhatsApp ou SMS

Étape 3 : Cotiser ou recevoir

1. Dans les détails de la tontine → "Cotiser maintenant"
2. Sélectionnez MTN MoMo ou Orange Money
3. Payez le montant indiqué

Quand c’est votre tour → vous recevrez le montant total collecté !

Étape 4 : Suivre et gérer

1. Dashboard → voir toutes vos tontines
2. Notifications → rappels automatiques
3. Reçus → historique des paiements
