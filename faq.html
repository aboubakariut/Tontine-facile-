Questions fréquentes

Comment créer une tontine sur Tontine Facile ?
Allez dans "Mes Tontines" → cliquez sur "+ Créer une nouvelle tontine". Remplissez le nom, le montant, la fréquence (hebdo/mensuelle), le nombre de membres et la date de début. Vous pouvez choisir MTN MoMo ou Orange Money comme moyen principal. Une fois créée, partagez le lien ou le code d'invitation avec vos contacts.
Comment payer ma cotisation ?
Cliquez sur "Cotiser maintenant" dans les détails de votre tontine. Vous serez redirigé vers MTN MoMo ou Orange Money (selon votre choix lors de la création). Le paiement est automatique et tracé. En cas de paiement manuel (espèces), le gestionnaire peut valider.
Que se passe-t-il si je suis en retard ?
Vous recevrez une notification de rappel 24h avant et le jour J. Après 48h de retard, une pénalité peut s'appliquer (définie par le gestionnaire). Le tour peut être décalé ou passer au suivant selon les règles de la tontine. Contactez votre gestionnaire pour régulariser.
Est-ce sécurisé ? Mes données et argent sont protégés ?
Oui ! Nous utilisons le chiffrement pour vos données. Les paiements passent par les API officielles MTN MoMo et Orange Money (pas de stockage de codes secrets). L'historique est immuable et visible par tous les membres pour plus de transparence. Nous ne partageons jamais vos infos.
Comment inviter des membres ou rejoindre une tontine ?
Pour inviter : partagez le lien/code depuis les détails de la tontine (via WhatsApp, SMS...). Pour rejoindre : cliquez sur le lien reçu ou entrez le code dans "Rejoindre une tontine" depuis le dashboard.
Que faire si je n'ai pas reçu mon tour ?
Vérifiez l'historique des transactions et le calendrier des tours. Contactez le gestionnaire de la tontine. Si problème technique, ouvrez un ticket via la section Support ci-dessous.
Comment changer mon numéro ou mot de passe ?
Allez dans Profil → "Modifier profil" ou "Changer le mot de passe". Pour le numéro, une vérification OTP est requise pour des raisons de sécurité.
L'application est-elle gratuite ?
Oui pour les fonctionnalités de base (création, gestion, paiements). Certaines options avancées (rapports PDF détaillés, support prioritaire) pourraient être payantes à l'avenir.
